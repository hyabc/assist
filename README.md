# Prerequisite

1. Add four environmental variable.

```
export amapkey="******"
export access_key_id="******"
export access_key_id="******"
export appKey="******"
```

2. Place libdarknet.a, yolov3.cfg, yolov3.weights under directory.

# Compilation

```
cmake .
make
```

# Exec

```
./spawn
```
